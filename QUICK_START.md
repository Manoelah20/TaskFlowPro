# 🚀 Quick Start - TaskFlow Pro

## 🎯 O que é o TaskFlow Pro?

Uma ferramenta **enterprise** de gestão de projetos com **13 funcionalidades avançadas** para organizar tarefas, colaborar em equipe e otimizar produtividade.

---

## ⚡ Começando em 5 Minutos

### 1. Acesse a Aplicação
```
Frontend: http://localhost:5173
Backend:  http://localhost:5000
```

### 2. Crie sua Primeira Tarefa
1. Preencha **"Nova tarefa"**
2. Adicione **descrição** (opcional)
3. Defina **prioridade** (Alta/Média/Baixa)
4. Configure **deadline** (opcional)
5. Estime **tempo** em horas (opcional)
6. Adicione **tags** separadas por vírgula (ex: `frontend,urgent`)
7. Atribua a um **usuário** (opcional)
8. Clique **➕ Adicionar Tarefa**

### 3. Trabalhe com o Kanban
- **To Do**: Tarefas para fazer
- **Doing**: Em andamento
- **Done**: Concluídas

**Arraste e solte** tarefas entre colunas!

## 🎮 Funcionalidades Essenciais

### 📊 Analytics (Botão 📊)
- Veja métricas de produtividade
- Analise tempo médio por tarefa
- Identifique padrões de trabalho

### 📅 Calendário (Botão 📅)
- Visualize tarefas por deadline
- Identifique tarefas atrasadas (borda vermelha)
- Planeje seu cronograma

### ⏱️ Time Tracking
- **▶️ Iniciar**: Clique para começar o timer
- **⏸️ Pausar**: Clique para parar
- **Visualizar**: Tempo aparece no card

### 🏷️ Tags e Filtros
- **Criar tags**: `frontend`, `backend`, `urgent`, `bug`
- **Filtrar**: Use os selects no header
- **Analytics**: Veja breakdown por categorias

## 🤖 Recursos Avançados

### 🤖 Automações (Botão 🤖)
- **Regras automáticas**: Configure triggers
- **Exemplo**: Mover tarefas atrasadas para alta prioridade
- **Como**: Ative/desative com toggle ✅/❌

### 📋 Templates (Botão 📋)
- **Sprint Development**: Cria 4 tarefas padrão
- **Como**: Selecione → "Criar do Template"
- **Benefício**: Padroniza processos

### 🔥 HeatMap (Botão 🔥)
- **Visualização**: Grid de atividade (90 dias)
- **Cores**: 🟢 baixa → 🟡 média → 🔴 alta
- **Uso**: Identifique padrões de produtividade

### 🔌 Integrações (Botão 🔌)
- **Slack**: Notificações no canal
- **GitHub**: Sync de issues
- **Calendar**: Sync de deadlines
- **Como**: Ative com toggle ✅/❌

### 🔔 Notificações (Botão 🔔)
- **Badge numérico**: Indica não lidas
- **Tipos**: 🔵 info, 🟢 success, 🟡 warning, 🔴 error
- **Ação**: Clique para marcar como lida

### 👥 Time Performance (Botão 👥)
- **Métricas individuais**: Por usuário
- **Eficiência**: Percentual de conclusão
- **Workload**: Carga vs disponibilidade

### 🔗 Dependências (Botão 🔗)
- **Visualização**: Veja tarefas que dependem de outras
- **Bloqueio**: Tarefas dependentes ficam bloqueadas
- **Planejamento**: Organize sequência de execução

| Problema | Solução |
|----------|---------|
| Tarefa não move | Verifique dependências 🔗 |
